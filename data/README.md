# Eye, scene and route data

This directory contains data required to run the simulations.

It contains:

* scene files - glTF encoded 3D models
* navmesh files - binary navigation meshes that match up to a glTF file
* eye files - CSV data (read by compound-ray) that specifies ommatidial position, direction and parameters
* route data - CSV files specifying an ant's 2D position
* camera positioning files - JSON files with 'film directions' for movie making
