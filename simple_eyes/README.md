# Eye generation programs

These programs were used to generate some of the .eye files in `data/eyes/`

**make_cyl_eye.cpp** generates the ommatidia that reproduce the view of a cylindrical panoramic eye, similar to the view used in many ant navigation studies.

**ant_eyes.cpp** generates some toy ant eyes with plausible size and spacing and a hexagonal layout (but *not* the biologically realistic eye that is presented in the paper).
